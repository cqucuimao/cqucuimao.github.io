Mac系统下搭建hexo博客

Github创建仓库

仓库名称必须为：username.github.io。如下图所示：

安装git/nodejs

安装步骤省略，安装完成后使用以下命令验证是否安装成功：

# 验证git
git --version

# 验证nodejs
node -v

安装hexo

执行下面的命令进行安装：

npm install -g hexo-cli
npm install hexo --save

创建博客文件夹，并初始化博客框架：

# 创建博客目录
mkdir hexo_blog

# 进入博客目录
cd hexo_blog

# 初始化
hexo init

# 启动服务
hexo s

出现以下内容说明启动成功：

浏览器访问：http://localhost:4000
